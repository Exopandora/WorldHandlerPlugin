If you want to run this in a dev environment then copy the source of WorldHandler in this folder so the structure is as follows:

YOUR_PROJECT/
└── src/
    ├── api/
    │   ├── java/
    │   │   └── WORLD_HANDLER_SOURCES
    │   └── resources/
    │       └── WORLD_HANDLER_RESOURCES
    └── main/
        ├── java/
        │   └── YOUR_SOURCES
        └── resources/
            └── YOUR_RESOURCES
